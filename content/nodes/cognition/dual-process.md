## node: dual-process
title: Dual Process Theory
tier: cornerstone
grammar: Paired duality

Two modes of thought run the mind: System 1 — fast, automatic, associative — and System 2 — slow, deliberate, effortful. System 1 operates by default; System 2 audits it, sometimes.

### Structure
- System 1 generates impressions continuously; System 2 endorses, corrects, or overrides them
- Cognitive load shifts reliance toward System 1 — tired minds default to shortcuts
- Expertise moves tasks from System 2 to System 1: the practiced becomes automatic
- The organizing frame for the rest of this domain — biases are System 1 output, restructuring is System 2 work

### Note
Strong as a framework, weaker as neuroscience: there are not literally two brain systems, and several specific demonstrations in Thinking, Fast and Slow (notably the priming chapter) failed replication. Kahneman publicly acknowledged the weakness.

### Bridges
- [dual-process-system1] The fast, automatic default.
- [dual-process-system2] The slow, deliberate auditor.
- [heuristics] Heuristics and biases are System 1's shortcuts made visible.
- [cbt] CBT is applied dual-process work: automatic thoughts audited by deliberate restructuring.
- [reappraisal] Reappraisal requires deliberate System 2 engagement.

### Sources
- Kahneman (2011). Thinking, Fast and Slow.
- Evans & Stanovich (2013). Dual-process theories of higher cognition.

## node: dual-process-system1
title: System 1
tier: cornerstone
grammar: Paired duality / System

Fast, automatic, and effortless. Operates by default, pattern-matching from experience.

### Structure
- Parallel processing — handles multiple inputs simultaneously
- Source of heuristics, emotional reactions, and intuition
- Generates quick answers; hands off to System 2 only when challenged
- Cognitive load pushes behavior further toward System 1

### Note
System 1 is not irrational — it is optimized for speed. Most decisions are System 1, and most are fine.

### Bridges
- [heuristics] The shortcuts System 1 runs on.
- [cbt-automatic-thoughts] Automatic thoughts are System 1 outputs about self, world, and future.
- [attribution-theory] The Fundamental Attribution Error is a System 1 heuristic for explaining people.

### Sources
- Kahneman (2011). Thinking, Fast and Slow.

## node: dual-process-system2
title: System 2
tier: cornerstone
grammar: Paired duality / System

Slow, deliberate, and effortful. Monitors System 1 output and intervenes when the stakes are high.

### Structure
- Sequential processing — one complex problem at a time
- Required for logic, planning, and overriding intuition
- Limited capacity — cognitive load depletes System 2 availability
- Expertise moves tasks from System 2 back to System 1

### Note
A strong organizing metaphor, not a literal two-brain model. Parts of the priming research popularized alongside it failed replication.

### Bridges
- [cbt-restructuring] Cognitive restructuring is System 2 auditing System 1.
- [reappraisal] Deliberate reinterpretation is System 2 work applied to emotion.

### Sources
- Evans & Stanovich (2013). Dual-process theories of higher cognition.
