# Content Structure

This folder holds the canonical content architecture for the psychology literacy app.

## Canonical files

- `domains.md` is the domain architecture overview.
- `layers/` contains the Layer 2 + 3 pages for each domain.
- `layers/personality.md` covers personality and individual differences.
- `layers/motivation.md` covers motivation and behavioral regulation.
- `layers/cognition.md` covers cognition and judgment.
- `layers/relationships.md` covers relationships and attachment.
- `layers/emotion.md` covers emotion and regulation.
- `layers/self-identity.md` covers self and identity.

## Tagging system

Use stable tags to keep content searchable and consistent.

- `domain/*` for top-level domain assignment
- `layer/*` for the theory layer a page belongs to
- `type/*` for structure type such as trait, process, dimensional, or social
- `status/*` for content maturity such as draft, reviewed, or archived

## Working rule

Keep each page focused on one conceptual unit. Prefer short, explicit labels over implied taxonomy.
